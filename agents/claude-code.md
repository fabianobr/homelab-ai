# Instruções para Claude Code / OpenCode

Use este projeto como fonte de verdade operacional.

Antes de modificar qualquer coisa:

1. Leia `INVENTORY.yaml`
2. Leia `SECURITY.md`
3. Leia `SERVICES.md`
4. Leia `STANDARDS.md`

Regras:

- Não exponha portas públicas.
- Não instale ferramentas redundantes.
- Não altere arquivos de infraestrutura sem atualizar documentação.
- Sempre preserve simplicidade.
- Prefira Docker Compose.
